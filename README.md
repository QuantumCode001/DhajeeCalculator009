📖 About Dhajee Calculator

Dhajee Calculator is a unique entertainment-based web app where users can calculate a funny “Dhajee Price” depending on different lifestyle choices and personality conditions.

Users can select options like:

🏠 House Type
👔 Job & Income
😎 Looks & Style
💪 Personality
🚗 Lifestyle
🔥 Attitude Level

Based on these selections, the app generates a funny money-based result instantly.

This project is created purely for fun, entertainment, and social enjoyment.
